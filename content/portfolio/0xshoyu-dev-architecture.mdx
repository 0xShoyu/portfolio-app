## üõ† Technical Deep Dive

### 1. Bleeding-Edge Styling: Tailwind v4 Native Cascades
Migrating to Tailwind v4 allowed me to discard the bloated `tailwind.config.ts`. I utilized the new `@theme` directive directly within native CSS to map design system variables. This Apple-inspired "Sky-Dark" theme relies heavily on glassmorphism and SVG noise filters to create a tactile UI.

```css
/* src/app/globals.css */
@theme {
  --color-background: #082f49; /* Sky 950 */
  --color-foreground: #f0f9ff;
  --color-primary: #38bdf8;    /* Sky 400 */
  --color-card: rgba(12, 74, 110, 0.5);
}
```

### 2. Interaction Design: The Radial Spotlight
To achieve a premium, physical feel without layout thrashing, I utilized Framer Motion's `useMotionTemplate` and `useMotionValue`. 

Instead of re-rendering the React tree on every mouse movement, the component tracks the cursor coordinates and updates a CSS radial gradient on a hardware-accelerated overlay.

```tsx
// GPU-Accelerated Hover State
<motion.div
  className="absolute -inset-px rounded-xl opacity-0 group-hover:opacity-100 z-10"
  style={{
    background: useMotionTemplate`
      radial-gradient(
        650px circle at ${mouseX}px ${mouseY}px,
        rgba(56, 189, 248, 0.1),
        transparent 80%
      )
    `,
  }}
/>
```
